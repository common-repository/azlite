Azlite is a small (but perfectly formed) WordPress plugin that allows you to add an Amazon RSS feed, tagged with your affiliate code to your blog.

== Installation  ==
Simply download the Zip-Archive and extract all files into your wp-content/plugins/ directory. Then go into your WordPress administration page, click on Plugins and activate it.

==  Useage ==
Enter the following in your post [azlite]keyword,results,aff[/azlite] where keyword is the Amazon result set you wish displayed, result is the number of results displayed and aff is your Amazon affiliate code.

Example:
[azlite]xbox 360,5,myaffid[/azlite]
